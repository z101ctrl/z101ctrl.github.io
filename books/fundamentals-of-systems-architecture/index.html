Fundamentals of Software Architecture: An Engineering Approach

By Mark Richards & Neal Ford · 2020 · ★★★★

Read this book mainly because I really wanted to understand how the microservices architecture worked. It has some nice metrics for measuring code modularity. Seems like a good reference book.

Main takeways are:

  • Take network reliability, latency, bandwith capabilities, network security, network distribution and transport costs into account when designing. Do not take anything for granted.
  • If you choose to have distributed systems that induces a team architecture also.
