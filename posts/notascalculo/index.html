Resumen cálculo numérico

Posted on Mar 18, 2018

Error de truncado

$$ \Tau = \frac{x\left( t+h \right) - x\left( t \right)}{h} - F\left(t,x\left( t \right) \right) = x’’\left( L \right) \frac{h}{2} $$

Teorema

  1. Dado $\Tau$ el error truncado.
  2. Si $\left| F\left( t,x \right) - f\left( t,y \right) \right| \le k\left| x-y \right| \implies max \left| x_i - x\left( t_{i} \right) \right| \le \frac{\Tau}{k} e^{k\left( T -t_0 \right) } $

Errores

  • Error absoluto: $\left| x - fl\left( x \right) \right|$
  • Error relativo: $ \left| \frac{x - fl\left( x \right) }{x} \right|$

Método de euler

$$ x_{i+1} = x_{i} + x_{i}‘h $$

Desigualdad de Gramall

$$ \left| x\left( t \right) \right| \le C_2 e^{C_1t} \forall t \in \left[ 0,T \right] $$

Clasificacion

  • Convergencia: $\lim_{n \to 0} = max\left| x_{i} - x\left( t \right) \right|$
  • Consistencia: $\lim_{n \to 0} \Tau_{max } = 0$.
